\section*{Abstract}

The rapid development of precision livestock farming has created a critical need for integrated systems capable of performing multiple concurrent health assessments. Traditional cattle monitoring research suffers from two primary limitations: addressing individual tasks in isolation, and relying on private datasets that prevent reproduction. This thesis proposes a unified, lightweight, spatiotemporal multi-task deep learning framework that simultaneously processes four monitoring tasks from image and video inputs: Body Condition Scoring (BCS), behavior recognition, lameness detection, and individual animal identification. Built exclusively on publicly available datasets, our model employs a shared EfficientNet-B0 encoder enhanced with a Convolutional Block Attention Module (CBAM) and Long Short-Term Memory (LSTM) sequence layers. Single-task baselines achieved strong results, including a BCS Mean Absolute Error (MAE) of 0.5566, behavior Macro F1-score of 0.7445, and 86.49\% individual identification accuracy. Frame-level lameness AUC reached 0.9829, but this is treated as an optimistic upper bound due to possible identity/background leakage on the 50-video dataset. However, when integrated into the unified multi-task model with hard parameter sharing, spatial tasks experienced severe negative transfer (BCS MAE degraded to 0.7827; behavior F1 degraded to 0.4948). Conversely, temporal modeling maintained performance. Lameness achieved 80.00\% uncalibrated sequence accuracy on the limited 50-video dataset; post-hoc threshold tuning produced perfect separation but is treated as invalid for final evaluation because it introduces test-set leakage. Meanwhile, identification improved to 97.58\% via temporal majority voting. While the unified framework theoretically reduces backbone memory duplication by approximately 75\% compared with deploying four separate backbone networks, the observed gradient conflicts dictate that it serves as an honest baseline characterizing negative transfer rather than a system ready for clinical deployment. This establishes an openly evaluated public-dataset multi-task benchmark for cattle health monitoring and negative-transfer analysis.
