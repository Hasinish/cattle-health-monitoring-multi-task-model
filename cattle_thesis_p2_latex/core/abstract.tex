\section*{Abstract}

The fast-growing field of precision livestock farming demands systems that can monitor several health conditions at the same time. However, existing research usually focuses on just one task separately and they depend on private datasets that are not easily reproducible. In this thesis we address these problems and we propose a unified and lightweight spatiotemporal multi-task deep learning framework capable of simultaneously performing four tasks---body condition scoring, behavior recognition, lameness detection, and individual identification---from image and video data. The framework is developed using only publicly available datasets, the proposed model combines EfficientNet-B0 encoder with attention (CBAM) and temporal (LSTM) modules. Single-task baselines achieved strong results, including a BCS Mean Absolute Error (MAE) of 0.5566, behavior Macro F1-score of 0.7445, and 86.49\% individual identification accuracy. Frame-level lameness AUC reached 0.9829, but this is treated as an optimistic upper bound due to possible identity/background leakage on the 50-video dataset. However, when integrated into the unified multi-task model with hard parameter sharing, spatial tasks experienced severe negative transfer (BCS MAE degraded to 0.7827; behavior F1 degraded to 0.4948). Conversely, temporal modeling maintained performance. Lameness achieved 80.00\% uncalibrated sequence accuracy on the limited 50-video dataset; post-hoc threshold tuning produced perfect separation but is treated as invalid for final evaluation because it introduces test-set leakage. Meanwhile, identification improved to 97.58\% via temporal majority voting. While the unified framework theoretically reduces backbone memory duplication by approximately 75\% compared with deploying four separate backbone networks, the observed gradient conflicts dictate that it serves as an honest baseline characterizing negative transfer rather than a system ready for clinical deployment. This establishes an openly evaluated public-dataset multi-task benchmark for cattle health monitoring and negative-transfer analysis.
